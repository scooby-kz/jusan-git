# stepik-id-printer (h1)

Это проект для вывода моего пресонального идентификатора на *образовательной платформе* `stepik.org`.

## Возможности (h2)

Данный проект имеет следующий функционал.

- Выводит в stdout уникальный идентификатор
- Без зависимостей.
- Простой.

## Запуск
```
bash ./script.sh
```

## Портирование (h2)

